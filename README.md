# Siropi Sisters
A simple, elegant website for Siropi Sisters, two sisters who make and sell Baklava and other Greek goods in Saint George Utah. Built using HTML, CSS. This site includes our menu, photos, hours and contact information. 

## Features 
- Home page with bakery introduction
- Menu page displaying pastries and prices
- Gallery page with images
- Contact page with hours and address
